## 1. 产品概述

ABC-Group 组会资料库网站，用于在线展示课题组的组会记录、论文汇报、研究计划等内容，部署于 GitHub Pages。
- 面向课题组成员及访客，提供直观、美观的组会资料浏览体验
- 替代纯 README 文档，以交互式网页形式展示时间线、论文卡片、汇报人信息等

## 2. 核心功能

### 2.1 功能模块

1. **首页**：Hero 区域 + 课题组简介 + 最新组会动态
2. **组会记录页**：按时间线展示所有组会，每期包含论文卡片、汇报人、关键词标签、资料链接
3. **研究方向页**：展示课题组研究方向与待讨论课题，支持进度追踪

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | Hero 区域 | 课题组名称、标语、动态背景效果 |
| 首页 | 最新动态 | 展示最近 1-2 期组会摘要卡片 |
| 首页 | 研究方向概览 | 以图标+文字展示核心研究方向 |
| 组会记录页 | 时间线 | 按日期纵向排列组会节点 |
| 组会记录页 | 论文卡片 | 每篇论文的标题、关键词标签、汇报人、文件链接 |
| 组会记录页 | 筛选功能 | 按关键词/汇报人筛选组会内容 |
| 研究方向页 | 方向列表 | 展示研究方向及详细描述 |
| 研究方向页 | 课题清单 | 待讨论课题的 checklist，支持勾选状态 |

## 3. 核心流程

用户访问网站 → 浏览首页了解课题组概况 → 进入组会记录页按时间线查看历次组会 → 点击论文卡片查看详情/下载资料 → 进入研究方向页了解课题进展

```mermaid
    "浏览组会概况" --> "研究方向"

```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝灰 (#1a1a2e) + 亮蓝 (#4361ee) 作为强调色，辅以暖白 (#f8f9fa)
- **按钮风格**：圆角胶囊按钮，hover 时微浮起 + 发光效果
- **字体**：标题使用 Noto Serif SC（学术感），正文使用 Noto Sans SC
- **布局风格**：左侧导航 + 右侧内容区，卡片式布局
- **图标风格**：Lucide 线性图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页 | Hero 区域 | 深色渐变背景、大标题、粒子/网格动效、CTA 按钮 |
| 首页 | 最新动态 | 水平滚动卡片、论文缩略信息、关键词标签 |
| 组会记录页 | 时间线 | 左侧日期节点 + 右侧卡片，交替排列，滚动渐入动画 |
| 组会记录页 | 论文卡片 | 圆角卡片、论文标题、关键词标签、汇报人头像占位、下载按钮 |
| 研究方向页 | 方向列表 | 图标 + 标题 + 描述的网格布局 |
| 研究方向页 | 课题清单 | 带勾选框的列表，已完成项划线 |

### 4.3 响应式设计

- 桌面优先设计，移动端自适应
- 移动端导航折叠为汉堡菜单
- 卡片布局在移动端从多列切换为单列
